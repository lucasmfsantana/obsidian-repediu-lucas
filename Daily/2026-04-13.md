### Cardápio.ai (17:05) — Autenticação API
Lucas Assis (Cardápio.ai) tirou dúvida sobre fluxo de autenticação: usa `clientId` e `clientSecret` para gerar `accessToken`, que é passado em cada requisição. Token expira — o tempo de expiração vem no campo `expiresIn` (em segundos) no response da geração do token.

> [!important]
> **Conhecimento operacional Cardápio.ai:** O campo `expiresIn` no response de autenticação retorna o tempo de vida do token em segundos. O token deve ser renovado antes de expirar. O `type` é `bearer`.
