## Problemas [[WhatsApp API oficial]]
- **Por que não pode usar tag de link no botão para redes?**
	- O template é vinculado ao número na Meta e o link do botão fica fixo no template. Não é possível substituí-lo dinamicamente pelo link de cada empresa da rede. Alternativas: inserir o link no corpo do texto (funciona) ou usar um número por loja, o que permite usar o botão normalmente.
- **Por que não pode usar dois links ou wa.me nas campanhas?**
	- A Meta bloqueia templates que contêm links wa.me. A solução será implementar a automação de resposta, que contorna essa limitação.
