## Contexto: por que tantos bugs de campanha

Os problemas recorrentes em campanhas (Bora Baião, Nuoro, Italin House) não são bugs isolados — são sintomas da **migração da API de WhatsApp não-oficial (Evolution/Baileys) para a API oficial da Meta**. Agravante: descoberta de rate limit por hora na API oficial. Dois projetos em andamento para resolver:

1. **Fila de aprovação via RabbitMQ** — campanhas passarão por fila antes de ir para aprovação da Meta, respeitando rate limits
2. **Limite semanal com balanço inteligente** — Fernando e Lemes redesenhando para priorizar envio para quem tem mais chance de comprar

Previsão: 2-3 meses para estabilização completa.
