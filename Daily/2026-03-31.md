Workaround aplicado: aguardar 30 minutos e tentar novamente. Bug escalado via [[RPD-1915]] para o Vagner.

> Bug de login associado à duplicidade de usuário no banco (mesmo email, conta deletada + ativa) gera o erro "Sequence contains more than one element". Isso é diferente da instabilidade do Firebase — são dois problemas distintos.
